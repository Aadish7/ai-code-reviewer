# AI Code Reviewer

AI-powered code review platform built with FastAPI, React, GitHub API, and Groq LLM.

## Features

- Review GitHub Pull Requests
- Review Entire Repositories
- AI-generated Code Quality Analysis
- Security Recommendations
- Performance Suggestions
- GitHub User Repository Search

## Tech Stack

- Python
- FastAPI
- React
- Vite
- GitHub REST API
- Groq LLM
